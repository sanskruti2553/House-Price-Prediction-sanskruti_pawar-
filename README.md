# House Price Prediction

## Overview

This project predicts house prices using Machine Learning techniques based on various housing features such as area, bedrooms, bathrooms, stories, parking, furnishing status, and amenities.

## Dataset

* Dataset Name: Housing.csv
* Records: 545
* Features: 13
* Target Variable: Price

## Tasks Performed

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Linear Regression Model
4. Random Forest Regressor Model
5. Data Visualization
6. Model Evaluation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Visualizations

* House Price Distribution Histogram
* Correlation Heatmap
* Actual vs Predicted Price Scatter Plot

## Conclusion

The project successfully predicts house prices using machine learning models. Random Forest Regressor provided better prediction accuracy compared to Linear Regression and demonstrated strong performance on the housing dataset.
